# 3D Integration and Interactive Perspective Experiments

## Renderer Upgrade and Core Inspiration
[cite_start]To successfully integrate a 3D model into a 2D game framework—where the gameplay mechanics are inspired by the Google Chrome offline dinosaur jumping game—I upgraded the canvas renderer from the default 2D environment to P3D.

## Dynamic Motion and Control
[cite_start]I utilized the `rotateZ()` function to enable the 3D model to dynamically generate a pitch angle based on the real-time movement speed of the mouse.

## Conceptual Framing
[cite_start]Since the underlying core logic relies on obstacle avoidance, allowing the user to pilot a 3D spaceship navigating through a continuous stream of procedurally generated code-data mountains serves as a highly appropriate presentation within the context of computer art.
